Rick and Matt's Code Resources!

Below are the best resources we could find for learning Web Development. Enjoy!

JavaScript

MDN Docs for JavaScript

A comprehensive guide to pretty much every aspect of JavaScript.

Ruby

Why's Poignant Guide to Ruby

Why's Poignant Guide To Ruby is a classic within the Ruby community.

Rspec Documentation

The official website for the Ruby testing framework Rspec.

HTML and CSS

MDN Docs on Web Development

MDN docs on the three pillars of web development: HTML, CSS, and JavaScript.

Git

Git Book

Git is the official manual for the version-control-system git.

Learn Enough Git To Be Dangerous

From the Learn Enough series of books, Michael Hartl's book is a solid introduction to git.
